# Backend Architecture

This folder contains the backend implementation for DisasterShield AI.

## Planned Stack:
-   Django / Django REST Framework
-   AI Risk Prediction Module
-   Alert Engine
-   Volunteer Feedback System
-   Resource Recommendation Engine

## The Backend Will Expose REST APIs For:
-   Risk predictions
-   Alert triggering
-   Model performance metrics
-   Volunteer reports
-   Resource allocation data
