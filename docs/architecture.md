# System Architecture

DisasterShield AI follows a modular microservice-inspired architecture:

1. Data Ingestion Layer
   - Weather APIs
   - Satellite Data
   - IoT Sensors

2. AI Risk Engine
   - Disaster probability prediction
   - Confidence scoring

3. Alert Engine
   - SMS Gateway
   - Voice Broadcast
   - Cell Broadcast

4. Dashboard Layer
   - Risk Monitoring
   - Alert Logs
   - Resource Allocation

5. Feedback Loop
   - Volunteer Ground Reports
   - Model Retraining Pipeline
